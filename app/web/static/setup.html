관리자 계정 설정

새 설치라 기본 관리자 계정(admin)이 시드 비밀번호로 만들어져 있습니다. 누구나 알 수 있는 기본값이라 그대로 두면 보안 문제 — 지금 바꿔주세요.

갤러리 상단의 사용자 메뉴에 표시됩니다. 비워두면 "관리자".
최소 8자.

사진 폴더 추가 (권장)

사진이 들어있는 폴더의 절대 경로 하나만 등록하면 스캔이 시작됩니다. 나중에 관리 UI에서 더 추가하거나 수정할 수 있어요. 건너뛰기해도 설정은 끝납니다.

사이드바·관리 화면에 표시될 짧은 이름.
Docker로 띄웠다면 보통 /photos (컨테이너 안 마운트 경로). 시놀로지에서 직접 운영이라면 /volume1/photo 같은 식.

ML 모델 다운로드 (선택)

객체 인식(YOLO)·이미지 임베딩(CLIP)·얼굴 인식(YuNet+SFace)에 쓸 모델 가중치를 받습니다. 합쳐서 약 80 MB, 빠른 망에서 1-2분. 지금 건너뛰면 자동 분류 기능이 비활성화되며 나중에 관리 UI에서 다시 받을 수 있어요.
