새 폴더 추가

경로 입력 안내
  • 도커: docker-compose.yml에서 마운트한 컨테이너 안 경로를 입력하세요 (기본 /photos, 추가 마운트면 /photos2 등). 호스트 경로(/volume1/photo)는 컨테이너에서 보이지 않습니다.
  • 시놀로지 직접 설치: 호스트 절대 경로 그대로 (/volume1/photo).
  • 상태가 접근 불가로 뜨면 폴더 권한 문제입니다 — Synology Photos가 만든 폴더는 보통 d---------+(ACL 전용)이라 컨테이너 UID로는 못 읽습니다. 호스트에서 sudo chmod 777 /volume1/photo 한 번이면 해결됩니다.

추가 후 “시험”(200장)으로 한 번 돌려보고, 문제가 없으면 limit을 비우고 “스캔”으로 풀스캔하세요.
